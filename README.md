Color palettes drawn from 1920s data visualizations in Cerîde-i Adliyye (Justice Gazette) published in Turkey. Thanks to [Casual Archivist](https://casualarchivist.substack.com/p/poetic-justice) for distributing these.

You can install the package using `devtools::install_github("cbergey/ceride.i.adliyye")` in R.
