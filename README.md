Color palettes drawn from 1920s data visualizations in Cerîde-i Adliyye (Justice Gazette) published in Turkey. Thanks to [Casual Archivist](https://casualarchivist.substack.com/p/poetic-justice) for distributing these. This package was heavily based on Jilly MacKay's [tutorial for creating ggplot color palette packages](https://jillymackay.com/post/ggplot2-palettes-tutorial/).

You can install the package using `devtools::install_github("cbergey/ceride.i.adliyye")` in R.
